# BHE be v2

UI for BHE be

## Getting Started

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Env

```
API_URL
COPYRIGHT
CLIENT_SECRET
CLIENT_ID
NODE_ENV
LOCALE_PASS
APP_TITLE
```

## Link

```
node .\scripts\prelink.js
yarn link @bheui\components
```
